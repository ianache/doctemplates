## 🔒 Security Gate Integration

1. **Authenticated Shell:** All administrative and authoring UI endpoints strictly check OIDC credentials.
2. **HMAC Public Signing:** Public sharing uses cryptographically signed URLs to allow secure downloading without authentication, while logging events to the tracelog timeline.
3. **Sandbox Hardening:** Runtime proxies override `__getattribute__` to prevent malicious dunder injection and isolate Python execution environment.
