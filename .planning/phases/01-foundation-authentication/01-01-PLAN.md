Model shapes (backend/app/models/user.py, backend/app/models/session.py) that later plans (user_service, session_service, dependencies) import directly:
```python
class User(Base):
    __tablename__ = "users"
    id: Mapped[uuid.UUID] = mapped_column(primary_key=True, default=uuid.uuid4)
    sub: Mapped[str] = mapped_column(unique=True, index=True)
    email: Mapped[str]
    created_at: Mapped[datetime] = mapped_column(server_default=func.now())

class Session(Base):
    __tablename__ = "sessions"
    id: Mapped[str] = mapped_column(primary_key=True)   # opaque token, e.g. secrets.token_urlsafe(32)
    user_id: Mapped[uuid.UUID] = mapped_column(ForeignKey("users.id"))
    created_at: Mapped[datetime] = mapped_column(server_default=func.now())
    expires_at: Mapped[datetime]
    user: Mapped["User"] = relationship()
```

Test fixtures (backend/tests/conftest.py) that ALL later test files import implicitly via pytest fixture injection:
```python
def db_session() -> Session            # yields a SQLAlchemy Session against test_database_url, rolled back after each test
def client(db_session) -> TestClient   # FastAPI TestClient with get_db dependency overridden to use db_session
def rsa_keypair() -> tuple[bytes, bytes]   # (private_pem, public_pem), generated once per test session
def mint_test_jwt(rsa_keypair) -> Callable[[dict], str]   # signs a JWT with the test private key, kid="test-key-1"
def mock_jwks_client(monkeypatch, rsa_keypair)   # monkeypatches app.auth.jwks._get_jwks_client to return a stub whose get_signing_key_from_jwt(token) returns the test public key regardless of token, for any kid
```
