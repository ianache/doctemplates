# Phase 9: Search Documents Library & Audit Trace - Context

**Gathered:** 2026-07-09
**Status:** Ready for planning

<domain>
## Phase Boundary

Implement the Document Library backend and frontend. The backend will support querying and filtering generated issuances, fetching metadata and PDF files, verifying signatures on shared document URLs, and writing/listing chronological audit logs (tracelogs). The frontend will introduce a dedicated "Documents Library" navbar option, a search filter page, a detailed preview panel with an embedded PDF viewer, a download option, and an interactive audit log timeline.

</domain>

<decisions>
## Implementation Decisions

### Audit Tracelog Model
- **D-01 (Dedicated Audit Tracelog Table):** Create a dedicated SQLAlchemy model `DocumentTracelog` in `backend/app/models/document_tracelog.py`.
  - Columns:
    - `id`: UUID (Primary Key, default uuid4)
    - `issuance_id`: UUID (Foreign Key to `document_issuances.id` with cascade on delete)
    - `event_type`: String (Enum or exact values: `generation`, `download`, `share`)
    - `user_id`: UUID (Foreign Key to `users.id`, nullable for anonymous share downloads)
    - `metadata`: JSON (Holds client details: IP, user agent, etc.)
    - `created_at`: DateTime (server-side default func.now())
  - Add Alembic migration to instantiate the table.

### Cryptographically Signed Share Links
- **D-02 (HMAC Signed Share Links):** Document sharing will generate a cryptographically signed public link to prevent ID enumeration/scraping.
  - Endpoint: `GET /api/public/document-issuances/{id}/download?signature={hmac_signature}`
  - Signature generation/verification uses HMAC-SHA256 with the backend's configured `SECRET_KEY`.
  - Accessing the public download endpoint registers a `download` event in `document_tracelogs` with `user_id = None`.

### Frontend Navigation Layout
- **D-03 (Sidebar Documents Library Option):** Add a top-level navbar link "Documents Library" (`/document-issuances`) in `AuthenticatedShell.tsx` using the `description` or `folder_open` material icon.
  - This separates the assets/design configuration space from operational issuance outputs.

### the agent's Discretion
- The exact layout design of the Tracelog Timeline component in the UI.
- The structure of the public share URL generated by the frontend.
- Signature expiration rules (none required for MVP, but link generation can support optional duration).

</decisions>
