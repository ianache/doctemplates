<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser -> Preview API | User-edited mock JSON crosses into backend PDF rendering and validation. |
| PDF Blob -> Browser iframe | Backend-generated PDF bytes are displayed in the browser. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-03 | Tampering | MockDataPanel | mitigate | Parse JSON before submit and send only objects to previewDocumentDesign; display backend validation errors without bypassing backend validation. |
| T-10-04 | Information Disclosure | PreviewFrame | mitigate | Revoke object URLs on cleanup and keep preview behind authenticated apiFetch endpoint. |
| T-10-05 | Elevation of Privilege | previewDocumentDesign | mitigate | Use existing authenticated apiFetch; do not create public preview URLs or store credentials in client state. |
| T-10-SC | Tampering | npm installs | mitigate | No package installs are planned; keep package-lock unchanged. |
</threat_model>
